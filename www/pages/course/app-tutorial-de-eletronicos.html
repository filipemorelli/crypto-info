Tutorial de eletrônicos

O Electron é uma biblioteca de código aberto desenvolvida pela GitHub para a construção de aplicativos de desktop multiplataforma com HTML, CSS e JavaScript. O Electron consegue isso combinando Chromium e Node.js em um único tempo de execução e os aplicativos podem ser embalados para Mac, Windows e Linux.

Público

Este tutorial é projetado para aqueles aprendentes que aspiram a criar aplicativos de desktop multiplataforma para Linux, Windows e MacOS.

Pré-requisitos

Antes de continuar com este tutorial, você deve ter uma compreensão básica de Javascript (ES6) e HTML. Você também precisa saber sobre algumas API Node.js nativas, como gerenciamento de arquivos, processos, etc.
