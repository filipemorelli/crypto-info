Electron - Instalação

Para começar a desenvolver o uso do Electron, você precisa ter Node e npm (gerenciador de pacotes de nodejs) instalados. Se você ainda não possui estes, vá para a configuração do nodejs para instalar o nodejs em seu sistema local. Confirme se o nodejs e a npm estão instalados executando os seguintes comandos em seu terminal.

node --version
npm --version

O comando acima gerará a saída a seguir -

v6.9.1
3.10.8

Sempre que criamos um projeto usando npm, precisamos fornecer um arquivo package.json , que possui todos os detalhes sobre nosso projeto. npm facilita a configuração desse arquivo. Vamos configurar nosso projeto de desenvolvimento.

  • Aumente o seu terminal / cmd, crie uma nova pasta chamada hello-world e abra essa pasta usando o comando cd.

  • Agora, para criar o arquivo package.json usando npm, use o seguinte comando.

npm init

  • Ele irá pedir-lhe as seguintes informações -

Basta pressionar Enter e digite seu nome no campo "nome do autor".

Crie uma nova pasta e abra-a usando o comando cd. Agora execute o seguinte comando para instalar o Electron globalmente.

$ npm install -g electron-prebuilt

Uma vez que ele é executado, você pode verificar se o Electron está instalado do jeito certo executando o seguinte comando -

$ electron --version

Você deve obter o resultado -

v1.4.13

Agora que configuramos o Electron, deixe-nos avançar para criar nosso primeiro aplicativo usando ele.
