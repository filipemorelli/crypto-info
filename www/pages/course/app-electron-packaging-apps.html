Electron - Packaging Apps

A embalagem e a distribuição de aplicativos são parte integrante do processo de desenvolvimento de uma aplicação de desktop. Uma vez que a Electron é uma plataforma de desenvolvimento de aplicativos de desktop multiplataforma, a embalagem e a distribuição de aplicativos para todas as plataformas também devem ser uma experiência perfeita.

A comunidade eletrônica criou um projeto, o pacote eletrônico que cuida o mesmo para nodejss. Ele nos permite empacotar e distribuir o nosso aplicativo Electron com pacotes específicos do sistema operacional (.app, .exe etc) via JS ou CLI.

Plataformas suportadas

O Packer Eletrônico é executado nas seguintes plataformas host:

  • Windows (32/64 bit)
  • OS X
  • Linux (x86 / x86_64)

Ele gera executáveis ​​/ pacotes para as seguintes plataformas-alvo -

  • Windows (também conhecido como win32, ambos 32/64 bits)
  • OS X (também conhecido como darwin) / Mac App Store (também conhecido como mas)
  • Linux (para arquiteturas x86, x86_64 e armv7l)

Instalação

Instale o embalador de elétrons usando -

# for use in npm scripts
$ npm install electron-packager --save-dev

# for use from cli
$ npm install electron-packager -g

Aplicativos de embalagem

Nesta seção, veremos como executar o packager a partir da linha de comando. A forma básica do comando é -

electron-packager <sourcedir> <appname> --platform=<platform> --arch=<arch> [optional flags...]

Isto irá -

  • Encontre ou baixe a versão correta do Electron.

  • Use essa versão do Electron para criar um aplicativo em <output-folder> / <appname> - <platform> - <arch>.

--platform e --arch podem ser omitidos, em dois casos. Se você especificar - em vez disso, serão criados os pacotes para todas as combinações válidas de plataformas / arquiteturas de destino. Caso contrário, será criado um único pacote para a plataforma / arquitetura host.
