On-line de construção eletrônica

A interface do usuário de aplicativos eletrônicos é construída usando HTML, CSS e JS. Então, podemos aproveitar todas as ferramentas disponíveis para desenvolvimento web front-end aqui também. Você pode usar as ferramentas como Angular, Backbone, Reagir, Bootstrap e Fundação, para criar aplicativos.

Você pode usar o Bower para gerenciar essas dependências front-end. Instale bower usando -

$ npm install -g bower

Agora você pode obter todas as estruturas JS e CSS disponíveis, bibliotecas, plugins, etc. usando bower. Por exemplo, para obter a versão estável mais recente do bootstrap, insira o seguinte comando -

$ bower install bootstrap

Isso irá baixar bootstrap em bower_components . Agora você pode fazer referência a esta biblioteca em seu HTML. Vamos criar uma página simples usando essas bibliotecas.

Vamos agora instalar jquery usando o comando npm -

$ npm install --save jquery

Além disso, isso será necessário no nosso arquivo view.js. Já temos uma configuração main.js da seguinte maneira:

const {app, BrowserWindow} = require('electron')
const url = require('url')
const path = require('path')

let win

function createWindow() {
   win = new BrowserWindow({width: 800, height: 600})
   win.loadURL(url.format ({
      pathname: path.join(__dirname, 'index.html'),
      protocol: 'file:',
      slashes: true
   }))
}

app.on('ready', createWindow)

Abra o arquivo index.html e insira o seguinte código nela -

<!DOCTYPE html>
<html>
   <head>
      <meta charset = "UTF-8">
      <title>Hello World!</title>
      <link rel = "stylesheet" 
         href = "./bower_components/bootstrap/dist/css/bootstrap.min.css" />
   </head>
   
   <body>
      <div class = "container">
         <h1>This page is using Bootstrap and jQuery!</h1>
         <h3 id = "click-counter"></h3>
         <button class = "btn btn-success" id = "countbtn">Click here</button>
         <script src = "./view.js" ></script>
      </div>
   </body>
</html>

Crie view.js e insira a lógica do contador de clique nele -

let $ = require('jquery')  // jQuery now loaded and assigned to $
let count = 0
$('#click-counter').text(count.toString())
$('#countbtn').on('click', () => {
   count ++ 
   $('#click-counter').text(count)
})

Execute o aplicativo usando o seguinte comando -

$ electron ./main.js
